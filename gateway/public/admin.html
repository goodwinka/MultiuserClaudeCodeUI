    Плагины Claude Code
    

      
      
    

  

  

    Устанавливаются через claude plugin install. Хранятся в /etc/claude/plugins/ — доступны всем пользователям.
    После установки/удаления нужно перезапустить сессии.
  

  

    
Загрузка…

  


  
  

    Marketplaces
    
  

  

    Источники плагинов. После добавления marketplace становятся доступны плагины из него.
  

  

    
Загрузка…

  


  
  

    MCP-серверы (плагины)
    
  

  

    MCP-серверы глобально доступны всем пользователям через settings.json → mcpServers.
    После добавления перезапустите сессии пользователей.
  

  

    
Загрузка…

  


  
  

    Глобальные агенты
    
  

  

    Агенты доступны всем пользователям через ~/.claude/agents/ (символическая ссылка на /etc/claude/agents/).
    Для использования в Claude Code: /agents.
  

  

    
Загрузка…

  


  
  

    Глобальные скиллы
    
  

  

    Скиллы доступны всем пользователям через ~/.claude/skills/ (символическая ссылка на /etc/claude/skills/).
    Для использования в Claude Code: /skill-name.
  

  

    
Загрузка…

  


  
  

    Настройки Claude (settings.json)
    
  

  

    Глобальный файл /etc/claude/settings.json — настройки MCP-серверов, разрешений и переменных окружения для всех пользователей.
  


  
  

    Инструкции Claude (CLAUDE.md)
    
  

  

    Глобальный файл /etc/claude/CLAUDE.md — общие инструкции, правила и контекст для Claude Code. Доступен всем пользователям как ~/.claude/CLAUDE.md.
